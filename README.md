# 🚀 TechCareer Simulator

Um simulador interativo de carreira em tecnologia onde o usuário toma decisões profissionais e acompanha sua evolução ao longo do tempo.

---

## 📌 Sobre o Projeto

O **TechCareer Simulator** é uma aplicação web que simula a trajetória de um profissional de tecnologia.  
O usuário pode tomar decisões que impactam sua carreira, como aceitar propostas, estudar novas tecnologias e evoluir seu salário e nível.

Além disso, o sistema conta com um **ranking global**, permitindo comparar seu desempenho com outros usuários.

---

## 🎯 Funcionalidades

- 👤 Sistema de login de usuários
- 📊 Dashboard com status da carreira
- 💼 Simulação de decisões profissionais
- 📈 Evolução de salário e nível
- 🏆 Ranking de usuários
- 🎨 Interface moderna e responsiva
- ⚡ Atualização dinâmica de dados

### 🎨 Frontend
